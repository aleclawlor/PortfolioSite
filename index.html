Circa.Me

A social media application for Boston College Students that emphasizes making real-world social connections beyond the screen. The app uses graph algorithms to allow users to visualize their social network in a way that reflects their real world status. Mobile application written in React Native, while the backend is written in Node with MongoDB as a database.

SecureVision

A web application that uses license plate recongition to allow schools to track license plates of cars entering their schools. The application consists of three main components: a full-stack web application in React and Node, a Computer Vision pipeline in Python to segment license plates, and a Convolutional Neural Network in Pytorch for license plate character recognition.
